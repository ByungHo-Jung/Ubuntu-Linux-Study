명령어 히스토리/ 자동완성 기능 사용
alias 기능 사용
슈퍼 유저 권한 획득 방법
간단한 쉘 스크립트 작성

명령어
echo 화면에 텍스트를 출력(인수로 전달되는 텍스트/문자열을 표시하는데 사용)
whoami 현재 사용자명 출력
pwd : 현재 디렉토리 출력
cd: 디렉토리의 파일 목록 출력
ls: 디렉토리의 파일 목록 출력
cat: 파일 내용 출력
mkdir: 디렉토리 생성
touch: 빈 파일 생성
clear: 화면 정리

리다이렉트: 직접 출력을 파일 출력으로 변경

파이프: 명령의 실행결과를 다른 명령어의 인자로 넘기는 역할(|)
test.txt 파일에 
Linux
Hello
순으로 저장되어 있으면
ex) cat test.txt | sort -> 앞에 명령어는 Linux Hello를 출력하라는 말이고
sort를 거치면 Hello Linux 순으로 출력을 하게 된다.

alias-> 사용자가 자주 입력하는 옵션을 포함한 긴 명령어를 매번
입력하지 않고 짧은 문자열로 지칭할 수 있는 기능
(재부팅시 자동 등록 X)
alias: 등록된 alias 명령어들을 확인할 수 있음
alias 별칭='수행할 명령어' : 수행할 명령어를 별칭으로 지칭
unalias 별칭 : 해당 별칭 alias 삭제

sudo: root 권한을 얻기 위해 
sudo -i 또는 sudo -s: root 계정으로 전환 가능
 
su : 현재 계정을 로그아웃 하지 않고 다른 계정으로 전환
su -사용자 이름 : 사용자 이름으로 변경

logout이나 exit 명령어: 이전 계정으로 복귀

shell script(쉘 스크립트) : 파일 이용, 프로그램 실행, 문자열 출력에 이용

>>와 >의 차이
>>: 명령어 뒤에 나오는 파일에 추가  (기존의 내용에 추가됨)
>: 명령어 뒤에 나오는 파일에 쓸 때 사용(내용을 덮어쓰게 되게 됨)
